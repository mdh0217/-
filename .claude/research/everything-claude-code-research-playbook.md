# Everything Claude Code 리서치 플레이북

작업이 문서 중심이거나, 출처가 중요하거나, 광범위한 저장소 컨텍스트가 필요할 때 사용하세요.

## 기본 원칙

- 1차 문서와 직접 소스 링크를 우선적으로 사용하세요.
- 시간이 지나면 바뀔 수 있는 사실에는 구체적인 날짜를 포함하세요.
- 각 권고나 결론마다 짧은 근거 자료를 유지하세요.

## 권장 진행 순서

1. 먼저 로컬 코드와 문서를 검토하세요.
2. 불안정하거나 외부 사실에 대해서만 검색하세요.
3. 파일 경로, 명령어, 링크와 함께 결과를 요약하세요.

## 저장소 정보

- 주 언어: JavaScript
- 프레임워크: 감지되지 않음
- 감지된 워크플로우 수: 10개
