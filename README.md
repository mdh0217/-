# 🚀 Node.js 테스트 자동화 템플릿 (ECC Starter Kit)

AI 협업을 통해 구축한 **최적화된 프로젝트 구조**입니다. 반복적인 테스트 작업을 자동화하고, 실행 중 발생할 수 있는 논리적 오류를 방지하는 설계를 포함하고 있습니다.

## ✨ 주요 기능 및 특징
* **자동화된 테스트 러너:** `run-all.js`를 통해 프로젝트 내의 모든 테스트 파일을 한 번에 실행.
* **무한 루프 방지 (ECC Guard):** `stop.js` 훅 실행 시 발생할 수 있는 재귀 호출 문제를 `ECC_SKIP_TESTS` 환경 변수로 완벽 차단.
* **높은 안정성:** 현재 19개의 단위 테스트 케이스를 모두 통과(Pass)한 검증된 구조.
